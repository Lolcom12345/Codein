Basics of Html

Tags

1. < html > This is called Html tag. it represents the root of an Html document.

2. < head > This is called head tag. it includes different tags.

3. < title > This is called title tag. it is used to give name to our Website.

4. < body > This is called body tag. it cotains the main part of our Website.

5. < p > HTML paragraph or HTML p tag is used to define a paragraph in a webpage.
